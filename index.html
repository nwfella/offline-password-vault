About This Tool

Offline Password Vault + Generator is a paranoid's password manager that never touches a network — except for the privacy-preserving HIBP k-anonymity API for breach checks (only the first 5 hex chars of your password hash are sent).

Features

  • Password generation — XKCD-style word phrases, random alphanumeric, PINs, pronounceable passwords
  • Strength meter — real-time entropy calculation (bits) with color-coded bar
  • Ephemeral vault — site/password combos stored in sessionStorage only
  • Encrypted export — AES-256-GCM via the Web Crypto API, zero dependency
  • Breach check — privacy-preserving k-anonymity lookup against HaveIBeenPwned
  • Copy to clipboard — one-click copy for any password
  • Zero data on disk — unless you explicitly export, nothing is ever stored

Entropy Model

Entropy is calculated as H = log₂(NL) where N is the character set size and L is the length. For XKCD phrases it's H = Lwords × log₂(dictionary_size). The meter thresholds: <30 bits · 30–50 · 50–70 · 70–100 · >100 bits
